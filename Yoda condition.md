Yoda conditions är en programmeringstil där två delar av ett uttryck är omvända. Ett Yoda condition placerar konstanten först och variabeln därefter.

Vanlig ordning:
```javascript
if (value == 42) {/*...*/}
```

Samma uttryck, men som ett Yoda condition:
```javascript
if (42 == value) {/*...*/}
```
